O que é Front-end ?

É uma abstração, simplificando o componente pelo fornecimento de uma interface “amigável”





Histórico

No meio dessa mudança toda, muitos front-ends não estavam acostumados a lidar com lógica e regras de negócio que agora estavam transbordando para o lado do client-side, exatamente por que a praia deles era escrever CSS e Javascript para interface.





Para quê serve?

é a utilizado para criar a interface entre o sistema e o usuário. é a primeira camada com a qual nos deparamos ao acessarmos um site, uma intranet ou mesmo um sistema web.







Frameworks vs Bibliotecas

É uma ferramenta que auxilia o programador a trabalhar o determinada linguagem.

Algumas vezes uma biblioteca é ou se torna tão complexa que acaba virando um framework.

A ideia da biblioteca é compartilhar soluções por meio de funções ou métodos.





HTML e CSS

O nome HTML vem do inglês: Hypertext Markup Language, em português seria: Linguagem de Marcação de Hipertexto. HTML serve para dar significado e organizar a informação dos websites.

Cascading Style Sheets (CSS) e em português é uma “folha de estilo” composta por “camadas” e utilizada para definir a apresentação (aparência) em páginas
